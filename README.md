# Omega Dracula Theme by Ratakor

To use this theme, add `THEME_NAME` and `THEME_REPO` arguments to your `make` command:
```bash
$ make THEME_NAME=omega_dracula THEME_REPO=https://github.com/Ratakor/Omega-Themes
```
